Privacy Policy for BlueOff Chat

Last updated: July 8, 2026

BlueOff Chat is an offline Bluetooth chat application that allows users to communicate with nearby devices without using the internet. This Privacy Policy explains how the app handles information.

Information We Collect

BlueOff Chat does not require users to create an account and does not collect personal information such as name, email address, phone number, or location for online tracking.

The app may request certain Android permissions only to provide its core features, such as Bluetooth communication, nearby device discovery, notifications, media sharing, microphone access for voice messages, and storage or photo access if the user chooses to send files or images.

How Information Is Used

Information entered or shared inside the app is used only to provide offline chat functionality between nearby devices. Messages and media are not uploaded to our servers.

Bluetooth and Nearby Devices

BlueOff Chat uses Bluetooth and nearby device features to discover and connect with nearby users. These features are required for offline communication and are not used for advertising or tracking.

Data Sharing

We do not sell, rent, or share user data with advertisers or third parties. Since BlueOff Chat is designed for offline communication, messages are exchanged directly between connected devices.

Data Storage

Messages or media may be stored locally on the user's device depending on app functionality. Users can delete app data through the app settings or Android system settings.

Children's Privacy

BlueOff Chat is not designed to knowingly collect personal information from children. If you believe that a child has provided personal information through the app, please contact us so we can help address the issue.

Security

We take reasonable steps to protect user information. However, Bluetooth communication depends on device-level security and user behavior. Users should only connect with trusted nearby devices.

Changes to This Privacy Policy

We may update this Privacy Policy from time to time. Any changes will be posted on this page with an updated date.
